# Degeneracy

A Balatro mod adding a ton of new Ranks! With content primarily based off of the game Dungeons and Degenerate Gamblers

## Additions

As of latest dev build, Abandonia currently adds
* 23 Jokers
* 2 Decks
* 2 Blinds
* 1 Edition
* 1 Sticker
* Too many ranks to count!
* with more to come!
